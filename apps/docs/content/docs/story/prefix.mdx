但在下載時你可能會注意到許多字體名稱後面帶有特定的標示，例如 GB、TC、CL、MN、P、TC、Plus 等等。這些標示通常代表字體的語言、地區、用途或風格。

其中最常見的繁體中文後綴有：

* TC：參考早期鉛字印刷字形常見寫法。
* TW：按照台灣教育部的書寫風格。

你可以參考 emfont 作者毛哥EM的部落格的文章 [中文字體標示解析：字體中的 TC、TW、GB、P 是什麼意思？](https://emtech.cc/p/font-suffix)，裡面有完整的說明。
