你可以在任何框架中使用 emfont！以下是幾個小提醒：

1. 元素要保留 emfont 的 class。
2. 小元件局部渲染，可以使用 `root` 參數，只針對那塊重新掃描。
3. 動態更新內容後（比如聊天室、編輯器、動態清單）一定要自己再呼叫一次 `emfont.init({ root })`。


還是一句話，通常建議使用純 CSS 載入字體，詳見 [CSS 載入字體](/docs/css)。但這就很簡單了不用文檔了。
