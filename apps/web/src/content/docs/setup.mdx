<Steps>
    <Step>
    ## 挑選順眼的字型
    到[字型列表](https://font.emtech.cc/fonts)或 [emfont MCP](./mcp) 找到適合網站風格的字型，記下 ID。
    </Step>
    <Step>
    ## 設定 CSS
    以 jf open 粉圓 為例，它在 emfont 的字型 ID 是 `jfOpenHuninn` ，所以在 CSS 中這樣寫，
    ```css
    @import url("https://font.emtech.cc/css/jfOpenHuninn");

    body {
        font-family: "jfOpenHuninn", sans-serif;
    }
    ```
    如此一來整個 body 就會套用 jf open 粉圓字型
    > note: font-family 第二個參數為前一個字型不可用時的候補字型，你應該替換為接近預期風格的系統字型家族。

    </Step>

</Steps>

<Callout type=" ">記得使用字體 ID 而不是原始名稱。你有發現嗎，其實 CSS 存取位址就是你把字體詳情 URL 的 `/fonts` 改成 `/css`。</Callout>
