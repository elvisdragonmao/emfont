# 歡迎來到 emfont

## 免費為你的文字注入靈魂

文字是網站的靈魂，是思想的表達，更是美感的載體。然而，中文字體因為檔案龐大、需要耗費大量流量，很少有免費中文字體的 webfont 服務。emfont 的誕生正是為了填補這一空白，整理了許多開源免費的字體。為開發者提供一個優秀的中文字體解決方案。

## emfont 的特點

* **免費**：開源字體，完全免費。
* **簡單**：只需一點 CSS 或一行 JavaScript，自動完成所有操作。
* **極致壓縮**：字體壓縮為 `.woff2` 格式，載入速度快。
* **細微控制**：根據網站格式，自訂生成規則。
* **明確授權條款**：所有字體皆標示授權條款，避免侵權。
* **開源**：emfont 採用 Apache-2.0 授權。

身為一位全端工程師，emfont 在設計的時候希望能夠做到盡可能的簡單易用，同時又能夠滿足大部分的需求。我敢保證，你只需要一分鐘就能完全上手 emfont。
