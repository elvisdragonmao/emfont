	<Accordion title="emfont 要錢嗎？">
		不用。emfont 以 Apache 2.0 授權並完全開源於 GitHub，你可以的自由使用。
	</Accordion>
	<Accordion title="為甚麼我的字體沒辦法正常顯示？">
		* 請確保 emfont 的 JavaScript 有正確載入。
		* 確保你有使用 `emfont.init()` 初始化。
		* 確保你有在使用 `emfont.init()` 之前有先設定好 `emfont-` 的 class。
		* 如果是字母大小寫問題（如你使用 CSS 設定 `text-transform: uppercase;`），那麼你可以使用 `emfont.init({ caseSensitive: true })` 來初始化，或是創建一個隱形的 `span` 包住你要設定字體的元素，並且對那個 `span` 設定 `emfont-` 的 class。
		* 如果以上都不是，請到 [Discord 伺服器](https://dc.elvismao.com) 告訴我們，讓我們來幫你看看。
	</Accordion>
	<Accordion title="這些字型是否需要付費？">
		這些字型都是免費開源字型，不需要付費。
	</Accordion>
	<Accordion title="我可以在商業產品中使用 emfont 嗎？">
		可以。但請在使用前注意一下字體的授權條款。大多數的字體都是以 [SIL 開源字型授權 (OFL)](https://openfontlicense.org/) 授權，可以自由使用。
	</Accordion>
	<Accordion title="我可以在印刷品、網站、應用程式、刺青...上使用 emfont 嗎？">
		可以。但一樣請注意一下字體的授權條款。
	</Accordion>
	<Accordion title="我可以用 CSS 或 JavaScript 來對字體做立體、陰影、旋轉等效果嗎？">
		可以，我完全不在乎。唯一要注意的是 CSS 輪廓效果，因為可能會[輪廓重疊](https://jsbin.com/xewuzew/edit?html,output)，這種效果與可變字型不相容。
	</Accordion>
	<Accordion title="Normal 跟 Regular 兩個字重有什麼差別？">
	兩個通常都是指正常粗細（400）。不過如果你同時看到兩個的話（比如說 Noto 家族的字型） Normal 會比 Regular 細一點點，在 emfont 中這時會將 Normal 放在 350。
	</Accordion>

	</Accordion>

	</Accordion>
	<Accordion title="為什麼官網展示字型頁面有不一樣的句子？">
		對於一般的中文字型，預設都會是「義大利麵混 43 號混泥土」這句。不過，部分字型並非以中文為主要設計對象，或僅支援少量漢字，例如 Nerd Font、煮豆黑體等。因此，我們會依據各字型的實際支援範圍與應用情境，調整展示文字內容，讓呈現結果更貼近該字型的使用特性。
	</Accordion>
